# CardForge

Application web de création de cartes 3D personnalisables pour impression 3D.

## Installation

```bash
npm install
npm run dev
```

## Technologies

- Vite
- Three.js
- STL/OBJ Exporters

## Fonctionnalités

- Preview 3D temps réel
- Texte extrudé (relief/gravé)
- Dimensions personnalisables
- Export STL/OBJ
